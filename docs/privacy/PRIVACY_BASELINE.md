# Pilot Privacy Baseline

This document defines the pilot privacy baseline for the Texarkana 1885
prototype.

## Purpose

The prototype should be usable for a limited classroom pilot without collecting
student PII by default.

## Required Rules

- No unnecessary student PII should be collected.
- Student profiles should not be persisted in the prototype.
- Teacher authority remains the final instructional authority.
- Pilot materials should explain AI limitations.
- If a pilot later requires student work storage, retention and deletion rules must be documented first.

## Required Pilot Notes

Pilot materials should explain:

- what data is collected;
- why it is needed;
- who can access it;
- how long it is retained;
- and how deletion requests are handled.

## Current Texarkana Status

The current repo seeds the privacy baseline only. It does not yet implement
student rostering, production analytics, or a live retention/deletion system.
