# HQIM Framework

This document defines the minimum HQIM expectations for The Mind's Eye mission
design.

The product is not allowed to drift into generic game content that happens to
mention history.

## Purpose

HQIM requirements exist so that:

- historical evidence remains central;
- teacher intent drives the mission;
- student work produces explainable mastery evidence;
- and gameplay supports instruction instead of replacing it.

## Non-Negotiable HQIM Expectations

Every classroom-facing mission should eventually provide:

- a clear instructional purpose;
- source-grounded knowledge building;
- teacher-visible evidence and citations;
- a meaningful student task requiring reasoning, writing, discussion, or
  problem solving;
- and a mastery artifact the teacher can evaluate.

## Required Mission Components

Each reviewed mission should identify:

- the historical topic or problem;
- the source packet or cited claims used;
- the student-facing objective;
- the evidence students must inspect;
- the artifact or performance expected;
- the teacher review notes;
- the fictional gameplay frame if any;
- and the standard or standards the teacher selected.

## HQIM Boundary Rule

Gameplay framing may motivate the task, but it must not replace the
instructional core.

Examples:

- a missing ledger may frame the mission;
- but students still need to inspect evidence, reason about sources, and
  produce a meaningful artifact.

## Evidence Use Rule

Mission success should depend on evidence use, not only exploration.

Students should eventually have to:

- compare claims;
- distinguish verified fact from inference;
- explain uncertainty;
- and support conclusions with cited historical evidence.

## Teacher Authority Rule

The teacher remains the final authority over:

- mission approval;
- standards selection;
- source sufficiency;
- and final classroom use.

No autogenerated mission should bypass that review.

## Artifact Rule

Every reviewed mission should culminate in a student artifact or performance
that maps back to instructional goals.

Examples:

- editorial;
- witness statement;
- lab report;
- council proposal;
- oral briefing;
- or evidence-backed reflection.

## Current Texarkana Status

The current repo only seeds the HQIM framework and instructional contract.

It does not yet claim:

- final reviewed TEKS selection;
- rubric-ready artifact scoring;
- or production-ready standards alignment automation.
