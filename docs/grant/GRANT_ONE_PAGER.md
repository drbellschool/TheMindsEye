# Grant One-Pager

## Project

The Mind's Eye is a provenance-aware AI historical simulation platform that
helps teachers generate standards-aligned, evidence-based learning missions
from local primary sources.

## Why It Matters

Students should not just consume historical content. They should reason with
evidence, distinguish verified fact from inference and gameplay framing, and
produce teacher-reviewed artifacts that support real instruction.

## What The Prototype Proves

The Texarkana 1885 vertical slice is designed to prove a repeatable town-package
engine with:

- map-linked historical evidence;
- explicit provenance labels;
- teacher review and approval;
- standards alignment;
- readable classroom mission flows;
- and visible student-facing support contracts.

## Technical Novelty

The value is not simply that AI can write a mission.

The system combines:

- town-package architecture;
- map-linked historical simulation;
- claim-level provenance;
- confidence-labeled inference;
- teacher-directed mission generation;
- and explainable classroom evidence.

## Pilot Value

The pilot-ready direction is useful for teachers because it can:

- keep historical uncertainty visible;
- support mission planning from local source material;
- preserve teacher authority;
- and produce an instructional path that is narrow enough to review.

## Non-Dilutive Funding Fit

The project is structured for non-dilutive support because it emphasizes:

- educational impact;
- historical rigor;
- repeatable architecture;
- and careful handling of privacy, IP, and source rights.

## Current Status

This one-pager is a framing document, not a commitment to expand the prototype
beyond Texarkana 1885. The next step after this is pilot partner planning, not
broader product expansion.
